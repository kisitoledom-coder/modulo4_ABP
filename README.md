## Descripción
Aplicación en JavaScript desarrollada paso a paso que permite realizar operaciones matemáticas básicas a través de la consola del navegador. 

## Funcionalidad
El proyecto utiliza un objeto principal (`calculadora`) que encapsula métodos matemáticos y almacena un historial de operaciones. El usuario interactúa mediante `prompt` para ingresar números y elegir la operación (suma, resta, multiplicación, división). La aplicación utiliza un ciclo `while` para mantenerse en ejecución hasta que el usuario decida salir, y finalmente imprime un historial detallado en la consola recorriendo un arreglo de objetos mediante `forEach()`.
